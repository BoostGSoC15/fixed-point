Macro BOOST_FIXED_POINT_ENABLE_GMP_BACKENDS

BOOST_FIXED_POINT_ENABLE_GMP_BACKENDS — When this option is defined, fixed-point uses multiple-precision backends (when required) based on GMP instead of Boost.Multiprecision's self-written binary floating-point backend. GMP backends use highly optimized assembler and advanced multiplication algorithms that are significantly faster for high digit counts in excess of a few hundred binary digits.

Synopsis

// In header: <boost/fixed_point/fixed_point.hpp>

BOOST_FIXED_POINT_ENABLE_GMP_BACKENDS
