Header <boost/fixed_point/fixed_point_negatable_next.hpp>

Definitions of fixed_point functions for representation distance & finding adjacent fixed-point values.

See Also:

http://www.boost.org/doc/libs/release/libs/math/doc/html/math_toolkit/next_float.html Function fixed_distance finds the number of gaps/bits/ULP between any two fixed_point values.

If the fixed_point values are viewed as the underlying integral representation type, then their difference is the number of ULP/gaps/bits different.
