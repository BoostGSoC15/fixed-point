Header <boost/fixed_point/fixed_point_negatable.hpp>

Fixed_point negatable class used for signed fractional arithmetic.

This is a partial reference implementation for the proposed by Lawrence Crowl, "C++ binary fixed-point arithmetic" as specified in N3352.
In this file, we implement a prototype for the proposed negatable template class.
(See fixed_point_nonnegative.hpp for an unsigned version).

See Also:

http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2012/n3352.html
