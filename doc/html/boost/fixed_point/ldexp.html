boost::fixed_point::ldexp

Synopsis

// In header: <boost/fixed_point/fixed_point_negatable.hpp>


template<const int IntegralRange, const int FractionalResolution, 
         typename RoundMode, typename OverflowMode> 
  negatable< IntegralRange, FractionalResolution, RoundMode, OverflowMode > 
  ldexp(negatable< IntegralRange, FractionalResolution, RoundMode, OverflowMode > x, 
        int exp2);

Description

std::ldexp function <cmath> implementation for negatable type.
Multiplies a floating point value x by the number 2 raised to the exp2 power.
TBD examples.

Parameters:

exp2

power of 2 to use to multiply.

x

Fixed-point value to multiply by exp2 integral power of 2.

Returns:

x * 2^exp2.
