Function template fixed_advance

boost::fixed_point::fixed_advance

Synopsis

// In header: <boost/fixed_point/fixed_point_negatable.hpp>


template<const int IntegralRange, const int FractionalResolution, 
         typename RoundMode, typename OverflowMode> 
  negatable< IntegralRange, FractionalResolution, RoundMode, OverflowMode > 
  fixed_advance(negatable< IntegralRange, FractionalResolution, RoundMode, OverflowMode > x, 
                typename negatable< IntegralRange, FractionalResolution, RoundMode, OverflowMode >::value_type distance);

Description

Function fixed_advance advances a fixed_point number by a specified signed number of ULP (https://en.wikipedia.org/wiki/Unit_in_the_last_place), incrementing (or decrementing) the integral type value by distance bits.

Warning

It is not possible to advance more than the maximum value of the integral representation type in a single call of fixed_advance.
