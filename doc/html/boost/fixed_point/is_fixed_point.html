Struct template is_fixed_point

boost::fixed_point::is_fixed_point

Synopsis

// In header: <boost/fixed_point/fixed_point_negatable.hpp>

template<typename T> 
struct is_fixed_point : public false_type {
};

Description

Compile-time querying of whether or not a given type T is fixed_point.

Example:

std::numeric_limits<negatable<7,-8>>::is_signed == true
Note

Use std::numeric_limits<T>::is_signed to test if type T is a signed fixed_point type like negatable rather than an unsigned fixed point type like nonnegative.

Example:

std::numeric_limits<negatable<7,-8> >::is_signed == true
