Description

This is a partial reference implementation for the proposed by Lawrence Crowl, "C++ binary fixed-point arithmetic" as specified in N3352.
In this particular file, we implement a prototype for the negatable template class.
Examples:
boost::fixed_point::negatable<2,-5> x; // 8-bit
negatable<15, -16> y; // 32-bit
negatable<10, -53> y; // 64-bit
negatable<10, -245> y; // 256-bit (requires use of cpp_bin_float)

Note

Not all rounding or all overflow modes proposed in N3352 are yet implemented.

See Also:

http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2012/n3352.html
