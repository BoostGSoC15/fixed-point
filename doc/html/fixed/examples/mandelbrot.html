Example of computing images of a Mandelbrot set at low and high-resolution

This section presents examples of calculating the famous Mandelbrot set at low and high resolution.

This shows a Mandelbrot set generated at quite low-resolution of 160x128 pixels using a 32-bit fixed-point type.

Additional images having millions of pixels showing different areas of the Mandelbrot set are at: aqua full, yellow top, swirl, black and white seahorses.

These images have been be generated at high resolution using boost::fixed_point::negatable<16, -111> (fits in a 128-bit integer type). Even higher resolution is only limited by ones computer memory and patience for the long calculation time.

Full code is at fixed_point_mandelbrot.cpp. The program uses Boost.Gil for graphical representation of jpeg files and needs to be linked with the jpeg-6b library. The code uses neither multithreading nor a GPU. Calculating the Mandelbrot iterations in the separate rows of the image, however, could be parallelized in a straightforward fashion, at the cost of using more memory.
