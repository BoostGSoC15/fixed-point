Constructing and Interconverting Between Number Types

All of the fixed-point types in the Boost.Fixed_point Library have certain conversion rules in common.

Construction and assignment of a fixed-point type from another fixed-point type having the same range and resolution simply uses copy semantics.

Construction and assignment of a fixed-point type from another fixed-point type having more resolution uses truncation and rounding of the resolution.

Construction and assignment of a fixed-point type from another fixed-point type having more range uses truncation of the range.

If the other fixed-point type has more range and more resolution, the the range is truncated and the reolution is truncated with rounding.
