Get and Set functions ?

There are interface functions to access the private representation field containing the underlying integral type used to hold the fixed_point value. 

const value_type&  representation() const; // Read-only access.
const value_type& crepresentation() const; // Read-only access.
      value_type&  representation()        // Read-write access.

For some purposes, being able to both set and get without any conversion is invaluable, (but clearly may become a potential source of error).
